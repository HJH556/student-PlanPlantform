# 基于AI的大学生职业规划智能体项目介绍

## 项目概述

本项目是一个基于AI的大学生职业规划智能体平台，旨在帮助大学生进行职业规划、岗位探索、简历分析等功能。项目采用前后端分离架构，前端使用Vue 3 + TypeScript，后端使用FastAPI + Python，集成了LangChain框架和通义千问大模型，提供智能化的职业规划服务。

项目经过持续优化和完善，现已具备完整的知识图谱系统、智能行动计划生成、多平台支持等先进功能，为大学生提供全方位的职业发展指导。

- **AI驱动**: 集成LangChain框架和通义千问大模型

- **数据可视化**: 丰富的图表展示能力分析和岗位匹配
- **知识图谱**: Neo4j图数据库构建职业关系网络
- **多平台支持**: PC端、移动端、小程序全平台覆盖

### 💡 功能特色
- 智能简历解析和内容提取
- 个性化职业发展路径规划

- 职业能力评估和提升建议
- 数据驱动的职业决策支持
- **知识图谱可视化**: 岗位关联关系和发展路径展示
- **智能行动计划**: 基于AI分析的个性化发展建议
- **用户隔离系统**: 多用户数据隔离和个性化展示
- **实时问答系统**: AI助手提供实时职业咨询

## 技术栈详情

### 前端技术栈
- **框架**: Vue 3 + TypeScript
- **UI组件库**: Element Plus
- **状态管理**: Pinia
- **路由**: Vue Router
- **数据可视化**: ECharts
- **文件处理**: Mammoth (Word解析) + PDF.js + Tesseract.js (OCR)
- **HTTP客户端**: Axios
- **构建工具**: Vite

### 后端技术栈
- **Web框架**: FastAPI
- **数据库ORM**: SQLAlchemy
- **AI框架**: LangChain
- **大模型**: 阿里云通义千问
- **认证**: JWT + bcrypt
- **数据库**: SQLite + Neo4j图数据库
- **文件处理**: Python-multipart

### 移动端技术栈
- **框架**: Uni-app (Vue 3)
- **跨平台**: 支持微信小程序、H5、App
- **状态管理**: Vuex
- **网络请求**: uni.request

## 系统架构设计

### 整体架构
```
前端 (Vue 3 + TypeScript)
    ↓ HTTP/HTTPS (RESTful API)
后端 (FastAPI + LangChain)
    ↓ ORM/API调用
数据库层 (SQLite + Neo4j)
    ↓ AI服务调用
AI智能体 (通义千问 + 自定义逻辑)
```

### 数据流设计
1. **用户输入** → 前端收集 → API调用
2. **后端处理** → 数据验证 → AI分析 → 结果返回
3. **前端展示** → 数据可视化 → 用户交互

### 安全架构
- JWT令牌认证
- 密码bcrypt加密
- API请求验证
- 文件上传安全控制

## 部署与运行

### 前端部署
```bash
cd 前端/vue-A13
npm install
npm run dev  # 开发环境
npm run build  # 生产环境构建
```

### 后端部署
```bash
cd 后端
pip install -r requirements.txt
uvicorn main:app --reload  # 开发环境
```

### 移动端部署
```bash
cd app端
npm install
# 使用HBuilderX进行打包发布
```
