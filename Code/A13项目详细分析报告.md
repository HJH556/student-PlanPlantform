# 基于AI的大学生职业规划智能体项目报告

## 一、项目概述

### 1.1 项目背景
本项目是一个基于AI的大学生职业规划智能体平台，旨在帮助大学生进行职业规划、岗位探索、简历分析等功能。项目采用前后端分离架构，前端使用Vue 3 + TypeScript，后端使用FastAPI + Python，集成了LangChain框架和通义千问大模型，提供智能化的职业规划服务。

### 1.2 项目版本
- **版本号**: v2.0 (知识图谱增强版)
- **最后更新**: 2025年4月7日
- **项目状态**: 功能完善
